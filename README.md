# **Monte Carlo Simulation Predicting Septa T2 Arrival Probability using real time transit data**

## __Motivation__
This project was done to see how much actual arrival times deviate from the estimate arrival times from transit apps. This project used a Monte Carlo Simulation to accurately predict the time of arrival using transit data in real time.

## __Sources__
Septa Transit View real time API

Septa GTFS static feed for scheduling

## __Status__
Actively collecting data since 07/03/26 using poller 

## __Project Technology Stack__
Python, request, pandas, SQLite/CSV
